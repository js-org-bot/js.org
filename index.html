Almost two decades after the domain js.org was registered for the first time, it finally serves its obvious purpose - the JavaScript community.

Right from the beginning opinions about JavaScript varied - some people love the language, others hate it. But there should be no doubt about the awesomeness of the community that made JavaScript one of the most popular programming languages.
In alignment to the free-and-open mindset of the community, JS.ORG made subdomains available for developers and their projects for free. Since this first step we also focused on tracking JavaScript trends and spreading news about it.

CONTRIBUTIONS ARE WELCOME: If you have the passion to start a collaborative project (e.g. learn.js.org), or if you want to be a member of and contribute to JS.ORG, feel free to contact me.

  • DNS

    If you are a JavaScript developer you may have already heard about - or even already use GitHub Pages to host your content. DNS.JS.ORG complements the free webspace provided by GitHub, with a free and sleek URL that fits your project.

  • STATS

    Ready for some number crunching? STATS.JS.ORG fetches statistical parameters for the most important JavaScript projects on GitHub and stuffs them in a giant table with 10k rows. This project aims to make trends of JavaScript visible.

  • NEWS

    You have something to share with the community or want to read about the stuff other JavaScript enthusisats do? NEWS.JS.ORG provides a way to easily spread, consume and discuss JavaScript projects.
